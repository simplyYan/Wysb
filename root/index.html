Wysb

Compiled multi-paradigm programming language designed to be fast, easy, lightweight, secure and broad.

About

Wysb is a general-purpose high-level programming language developed by Wesley Yan Soares Brehmer (who lives up to the name of the language, which is short for the author's name) to be an easy, fast, reliable, secure, broad and lightweight solution. Despite being high-level, Wysb has insane performance, reaching the same level as low-level languages.

Typing

Wysb has a nice strong and static typing, which allows for even more solid performance, along with other optimizations that make it a language with insane performance, achieving a level close to low-level languages.

Light, fast, secure

Wysb is designed to be quick, easy, light and safe. It offers native features for encryption and the creation of immutable variables. In addition, the language is extremely optimized and receives constant updates to improve performance and security. Our language can be as fast as low-level languages, if used in the right way and with good practices.

Ecosystem

Wysb has its own package manager, called "Cardwmy". With the package manager, you can use libraries created by the community, people just like you! Cardwmy allows you to download packages hosted on Github, and automatically configures them for you to use in your programs.

Easy

Wysb is extremely easy to read and learn. You can learn the basics of the language in a short time, and expand your knowledge by learning various libraries.

Outstanding features

Compiler

The Wysb compiler is innovative! It doesn't use a lexer or parser, it recognizes your code, searching for language patterns and loading the values, storing everything in a reusable temporary cache. The compiler is extremely light and fast, built with the Go language.

Memory allocation

In Wysb, memory allocation is not mandatory. You can choose in which situations you use it, although there are situations where it may be the best choice. Memory allocation allows for better performance and avoids errors.

Cross platform

Currently, Wysb is compatible with Windows, Linux and macOS, and in the future, it will be compatible with Android.

Areas that use Wysb the most

Back-end/web development (in general)

Machine learning/Deep learning

Development of desktop programs (Without and with GUI)

Data science

In addition to these areas, several other areas consume Wysb, as well as several large companies around the world. Everything that requires good performance, security, reliability and flexibility can be done with Wysb, that is, everything you can think of.
